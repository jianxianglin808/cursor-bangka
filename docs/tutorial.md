## 🎯 详细步骤

### 第一步：申请虚拟卡

#### 1.1 注册虚拟卡平台
- 选择可靠的虚拟卡服务商
- 完成实名认证（如需要）
- 充值账户余额

#### 1.2 开通虚拟卡
- 选择适合的卡类型（建议Visa/MasterCard）
- 充值足够金额（建议20-30美元）
- 记录卡号、有效期、CVV等信息

### 第二步：订阅Cursor Pro

#### 2.1 登录Cursor官网
```
https://cursor.sh/
```

#### 2.2 进入订阅页面
1. 点击右上角头像
2. 选择 "Upgrade to Pro"
3. 或直接访问订阅页面

#### 2.3 填写支付信息
- **卡号**：输入虚拟卡卡号
- **有效期**：MM/YY格式
- **CVV**：安全码
- **账单地址**：根据卡平台提供的地址填写

#### 2.4 完成支付
- 确认订阅信息
- 点击确认支付
- 等待扣款成功

### 第三步：验证订阅

#### 3.1 检查订阅状态
- 返回Cursor应用
- 查看账户状态
- 确认Pro功能可用

#### 3.2 测试功能
- 尝试使用GPT-4
- 测试代码补全
- 验证所有Pro特性

## ⚠️ 常见问题解决

### 问题1：支付失败
**原因：**
- 虚拟卡余额不足
- 卡片信息填写错误
- 地址信息不匹配
- 平台风控拦截

**解决方案：**
1. 检查卡内余额（建议多充5-10美元）
2. 仔细核对卡号、有效期、CVV
3. 确保地址信息与卡平台一致
4. 尝试更换浏览器或清除缓存
5. 联系虚拟卡平台客服

### 问题2：扣款成功但未激活
**解决方案：**
1. 等待5-10分钟
2. 退出并重新登录Cursor
3. 检查邮箱是否有确认邮件
4. 联系Cursor客服

### 问题3：卡片被拒绝
**解决方案：**
1. 确认卡片支持国际支付
2. 检查卡片是否被冻结
3. 尝试其他虚拟卡平台
4. 加入QQ群获取最新可用平台信息

### 问题4：续费问题
**注意事项：**
- 确保卡内余额充足
- 关注扣款日期
- 设置余额提醒
- 及时充值避免停用

## 💡 使用技巧

### 技巧1：成本优化
- 选择费率低的虚拟卡平台
- 合理充值，避免余额过多闲置
- 关注平台优惠活动

### 技巧2：安全建议
- 使用专卡专用（不要多个服务共用）
- 定期检查扣款记录
- 不要在虚拟卡中存放过多余额
- 保护好卡片信息

### 技巧3：稳定性保障
- 选择信誉良好的卡平台
- 避免频繁更换卡片
- 及时续费，保持服务连续性
- 加入社群获取最新信息

## 🔄 续费流程

### 自动续费
Cursor会自动从绑定的卡片扣款，确保：
- ✅ 卡内余额充足
- ✅ 卡片未过期
- ✅ 卡片状态正常

### 手动续费
如果自动续费失败：
1. 登录Cursor账户
2. 更新支付信息
3. 手动完成支付
4. 确认订阅恢复
